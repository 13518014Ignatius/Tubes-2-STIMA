FolderCrawler

Program ini merupakan program yang dapat melakukan pencarian suatu file pada sistem storage PC.
Program ini dapat melakukan pencarian dengan 2 algoritme: BFS dan DFS.
Program ini juga dapat melakukan pencarian untuk 1 file atau semua file dengan nama yang sama.
Untuk menggunakan program ini, diperlukan Visual Studio.

Program sudah dicompile dan dapat dijalankan di folder bin, tetapi bila ingin mengcompile program, dapat menggunakan cara berikut:
Menggunakan Visual Studio: Klik file .sln pada folder src, kemudian klik bar "Build" di bagian atas window, dan klik "Build program"

Cara menggunakan program menggunakan Visual Studio:
1. Setelah membuka file .sln, klik opsi "Start" (logo segitiga berwarna hijau yang memiliki nama project di sebelah kanannya) pada bagian atas window
2. Saat form muncul, pilih folder awal pencarian dengan klik "Change Folder..."
3. Isi nama file pada textbox di bawah tombol "Change Folder..."
4. Jika ingin mencari semua file dengan nama yang sama, tick pilihan "Find all occurences". Jika tidak, skip langkah ini.
5. Pilih algoritme yang digunakan, BFS atau DFS
6. Program akan menunjukkan graph dan path dari file yang dicari
